# Price Drop Checker

A Python Flask app that tracks Amazon product prices and sends email alerts.

## Setup

1. Clone the repo:
   git clone https://github.com/yourusername/price-drop-checker.git
2. Install dependencies:
   pip install -r requirements.txt  # or just Flask, requests, python-dotenv
3. Create a `.env` file with the following variables:
   SERPAPI_KEY=your_serpapi_key
   EMAIL_ADDRESS=your_email@gmail.com
   EMAIL_PASSWORD=your_gmail_app_password
4. Run the app:
   python app.py
